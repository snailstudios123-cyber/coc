ğŸ’¡ TECHNICAL EXPLANATION:
--------------------------

The spell list uses Unity's UI layout system:

1. PauseMenuManager.OpenSpellsMenu() calls PopulateSpellsList()

2. PopulateSpellsList():
   - Gets all spells from SpellManager
   - For each spell, instantiates SpellSlot prefab
   - Parents it to SpellsListContainer
   - Calls SpellSlotUI.Initialize()

3. VerticalLayoutGroup on container arranges children

4. ContentSizeFitter expands container to fit all children

5. ScrollRect allows scrolling if content too tall

If any step fails, list appears empty!

Most common issue: SpellSlot prefab doesn't have proper size,
so LayoutElement collapses to 0 height = invisible!
