ğŸ“Š TECHNICAL EXPLANATION
------------------------

Why icons might not show:

1. WRONG COLOR
   Unity UI Images multiply sprite texture by Image.color
   Gray/Transparent color = invisible sprite
   Solution: color = Color.white (1,1,1,1)

2. MISSING REFERENCES
   SpellSlotUI script can't find the Image component
   Solution: Reconnect references in prefab

3. LAYOUT ISSUES
   HorizontalLayoutGroup might be sizing icon to 0x0
   Solution: Set min/preferred sizes on LayoutElement

4. DISABLED GAMEOBJECT
   Icon GameObject might be inactive
   Solution: Ensure SetActive(true) when icon exists

5. NO SPRITE ASSIGNED
   Image component has no sprite reference
   Solution: SpellSlotUI.Initialize() sets icon.sprite
