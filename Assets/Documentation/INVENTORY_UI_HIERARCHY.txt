  INVENTORY UI - HIERARCHY REFERENCE
═══════════════════════════════════════════════════════════════════════════

This file shows the exact UI hierarchy you need to create for the Inventory
System to work properly. Follow this structure carefully!

═══════════════════════════════════════════════════════════════════════════
  FULL HIERARCHY
═══════════════════════════════════════════════════════════════════════════

Canvas
└── InventoryPanel (Panel) ← Add InventoryUI component here
    ├── Header (Panel - Optional)
    │   └── Title (TextMeshProUGUI: "INVENTORY")
    │
    ├── ItemsList (Scroll View)
    │   └── Viewport (Image with Mask)
    │       └── Content (RectTransform)
    │           ├── [Vertical Layout Group component]
    │           │   • Spacing: 5
    │           │   • Child Force Expand Height: OFF
    │           │   • Child Force Expand Width: ON
    │           │   • Child Control Height: OFF
    │           │   • Child Control Width: ON
    │           │
    │           └── [Content Size Fitter component]
    │               • Horizontal Fit: Unconstrained
    │               • Vertical Fit: Preferred Size
    │
    ├── ItemDetailPanel (Panel)
    │   ├── ItemIcon (Image)
    │   │   • Preserve Aspect: ON
    │   │
    │   ├── ItemName (TextMeshProUGUI)
    │   │   • Font Size: 24
    │   │   • Alignment: Center
    │   │
    │   ├── ItemDescription (TextMeshProUGUI)
    │   │   • Font Size: 16
    │   │   • Alignment: Left
    │   │   • Overflow: Overflow (for scrolling text)
    │   │
    │   ├── ItemTypeText (TextMeshProUGUI)
    │   │   • Font Size: 14
    │   │   • Text: "Type: Consumable"
    │   │
    │   ├── QuantityText (TextMeshProUGUI)
    │   │   • Font Size: 14
    │   │   • Text: "Quantity: 5"
    │   │
    │   ├── ButtonsPanel (Panel - Optional, for organizing buttons)
    │   │   ├── UseButton (Button)
    │   │   │   └── Text: "USE"
    │   │   │
    │   │   └── DropButton (Button)
    │   │       └── Text: "DROP"
    │   │
    │   └── [Initially set ItemDetailPanel to INACTIVE]
    │
    ├── FilterButtons (Panel - Optional)
    │   ├── AllButton (Button)
    │   │   └── Text (TextMeshProUGUI): "All"
    │   │
    │   ├── ConsumablesButton (Button)
    │   │   └── Text (TextMeshProUGUI): "Consumables"
    │   │
    │   ├── KeyItemsButton (Button)
    │   │   └── Text (TextMeshProUGUI): "Key Items"
    │   │
    │   └── EquipmentButton (Button)
    │       └── Text (TextMeshProUGUI): "Equipment"
    │
    └── BackButton (Button)
        └── Text (TextMeshProUGUI): "BACK"

═══════════════════════════════════════════════════════════════════════════
  ITEM SLOT PREFAB HIERARCHY
═══════════════════════════════════════════════════════════════════════════

ItemSlot (Button) ← Add ItemSlotUI component here
├── Icon (Image)
│   • Preserve Aspect: ON
│   • Color: (1, 1, 1, 0.5) - semi-transparent
│   • RaycastTarget: OFF
│
├── ItemName (TextMeshProUGUI)
│   • Font Size: 16
│   • Alignment: Left
│   • RaycastTarget: OFF
│
└── Quantity (TextMeshProUGUI)
    • Font Size: 14
    • Alignment: Right
    • Color: Yellow or White
    • RaycastTarget: OFF
    • Text: "x5"

LAYOUT SUGGESTION FOR ITEMSLOT:
• Width: 400-500 pixels
• Height: 60-80 pixels
• Icon: Left side, square (50x50)
• ItemName: Middle, flexible width
• Quantity: Right side, fixed width

═══════════════════════════════════════════════════════════════════════════
  RECOMMENDED LAYOUT POSITIONS
═══════════════════════════════════════════════════════════════════════════

InventoryPanel:
• Anchors: Stretch (fill screen)
• Offsets: Left: 0, Top: 0, Right: 0, Bottom: 0

ItemsList (Scroll View):
• Position: Left side of screen
• Width: 40-50% of screen
• Height: Most of screen height
• Example: PosX: 50, PosY: 0, Width: 400, Height: 600

ItemDetailPanel:
• Position: Right side of screen
• Width: 40-50% of screen
• Height: Most of screen height
• Example: PosX: 500, PosY: 0, Width: 400, Height: 600

FilterButtons:
• Position: Top of ItemsList
• Horizontal layout
• Each button: Width: 80-100

BackButton:
• Position: Bottom center or bottom left
• Width: 100-150
• Height: 40-50

═══════════════════════════════════════════════════════════════════════════
  QUICK CREATE SHORTCUTS
═══════════════════════════════════════════════════════════════════════════

IN HIERARCHY:
Right-click in Canvas →

Panel: UI → Panel
Button: UI → Button
Image: UI → Image
Text: UI → Text - TextMeshPro
Scroll View: UI → Scroll View

IMPORTANT SETTINGS:
• Scroll View → Delete Scrollbar Horizontal (optional)
• Scroll View → Vertical Scrollbar: Visibility: Auto Hide or Always Visible
• Content → Add Vertical Layout Group
• Content → Add Content Size Fitter

═══════════════════════════════════════════════════════════════════════════
  COLOR SCHEME SUGGESTIONS
═══════════════════════════════════════════════════════════════════════════

Background Panels:
• Main Panel: Dark semi-transparent (0, 0, 0, 0.8)
• Detail Panel: Slightly lighter (0.1, 0.1, 0.1, 0.9)

Buttons:
• Normal: (0.3, 0.3, 0.3, 1.0)
• Highlighted: (0.5, 0.5, 0.5, 1.0)
• Pressed: (0.2, 0.2, 0.2, 1.0)

Text:
• Headers: White (1, 1, 1, 1)
• Body text: Light gray (0.9, 0.9, 0.9, 1)
• Quantity: Yellow (1, 1, 0, 1)

Special:
• Consumable items: Green tint
• Key items: Gold tint
• Equipment: Blue tint

═══════════════════════════════════════════════════════════════════════════
  TIPS FOR FASTER SETUP
═══════════════════════════════════════════════════════════════════════════

1. Create one complete ItemSlot first, test it, then make it a prefab
2. Use Unity's UI Auto-Layout components (Layout Groups, Size Fitters)
3. Set anchors properly for responsive design
4. Test with different screen resolutions
5. Use TextMeshPro for better text quality
6. Preserve aspect ratio on item icons
7. Make panels semi-transparent so players can still see the game
8. Add subtle animations (optional) for button hovers
