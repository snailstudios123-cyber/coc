=======================================================================
QUICK START
=======================================================================

1. Go to: Tools → Setup Complete Snow Spell
2. Wait for the setup to complete
3. Press PLAY
4. Press Q to cast the snow spell!

=======================================================================
WHAT IT DOES
=======================================================================

The Snow Spell summons falling snow from the sky above the player:

✓ Snow falls from above in a wide radius
✓ Individual snowflakes drift down realistically
✓ Enemies touched by snow are FROZEN in place
✓ Frozen enemies turn BLUE
✓ Ice particles appear around frozen enemies
✓ Beautiful particle effects create blizzard atmosphere
✓ Lasts for 6 seconds with continuous snowfall
✓ Freezes enemies for 4 seconds

=======================================================================
HOW TO USE
=======================================================================

CONTROLS:
---------
• Press Q to cast the snow spell
• Costs 0.5 mana
• 2 second cooldown between casts

The snow will spawn above your player and rain down in a wide area,
freezing any enemies it touches!

=======================================================================
CUSTOMIZATION
=======================================================================

You can adjust these settings in the PlayerController component:

PLAYER SETTINGS:
---------------
• Snow Spell Key: Which key casts the spell (default: Q)
• Snow Spell Mana Cost: How much mana it costs (default: 0.5)
• Time Between Snow Cast: Cooldown in seconds (default: 2)

SPELL PREFAB SETTINGS (in FallingSnowSpell prefab):
---------------------------------------------------
• Damage: Damage per snowflake hit (default: 8)
• Freeze Duration: How long enemies stay frozen (default: 4 seconds)
• Snowfall Radius: Width of the snow area (default: 6 units)
• Snowfall Duration: How long snow keeps falling (default: 6 seconds)
• Spawn Height: How high above to spawn snow (default: 10 units)
• Snowflakes Per Second: Density of snowfall (default: 25)
• Snowflake Fall Speed: How fast snow falls (default: 2.5)

FREEZE EFFECT SETTINGS (in FreezeEffect.cs):
--------------------------------------------
• Frozen Color: The blue tint color for frozen enemies
• Show Freeze Particles: Toggle ice particle effects

=======================================================================
ADDING A SNOWFLAKE SPRITE
=======================================================================

For better visuals, add a snowflake sprite:

1. Find or create a snowflake sprite image
2. Import it into your project
3. Open the Snowflake prefab in /Assets/Prefabs/
4. Select the Sprite Renderer component
5. Drag your snowflake sprite into the Sprite field
6. Adjust the sprite size if needed

The spell works without a sprite (shows as a white circle), but looks
much better with a proper snowflake image!

=======================================================================
HOW IT WORKS TECHNICALLY
=======================================================================

COMPONENTS:
----------
1. SnowSpell.cs - Main spell controller
   • Spawns at player position
   • Creates snowflakes from above
   • Manages particle effects
   • Controls duration and area

2. Snowflake.cs - Individual snowflake behavior
   • Falls downward with drift
   • Sways side to side
   • Rotates while falling
   • Freezes enemies on contact

3. FreezeEffect.cs - Freeze status effect
   • Stops enemy movement (makes Rigidbody kinematic)
   • Stops animations (animator speed = 0)
   • Changes sprite color to blue
   • Disables enemy AI script
   • Creates ice particles
   • Auto-unfreezes after duration

SPELL FLOW:
----------
1. Player presses Q
2. PlayerController checks mana and cooldown
3. FallingSnowSpell spawns above player
4. Snowflakes spawn continuously for 6 seconds
5. Each snowflake falls with realistic motion
6. On contact with enemy:
   - Enemy takes damage
   - FreezeEffect component is added
   - Enemy freezes and turns blue
   - Ice particles appear
7. After 4 seconds, enemy unfreezes
8. After 6 seconds, spell ends

=======================================================================
CHANGING THE CAST KEY
=======================================================================

To change from Q to another key:

1. Select your Player GameObject in the scene
2. Find the PlayerController component
3. Look for "Snow Spell Settings"
4. Change "Snow Spell Key" to your desired key
   (e.g., KeyCode.E, KeyCode.R, KeyCode.Alpha1, etc.)

=======================================================================
TROUBLESHOOTING
=======================================================================

SPELL DOESN'T CAST:
------------------
• Check you have enough mana (0.5 by default)
• Wait for cooldown (2 seconds between casts)
• Make sure Snow Spell Prefab is assigned in PlayerController

ENEMIES DON'T FREEZE:
--------------------
• Make sure enemies have the "Enemy" tag
• Check that enemies have the Enemy.cs component
• Verify snowflakes have trigger colliders

NO VISUAL EFFECTS:
-----------------
• Particle system auto-creates if missing
• Add a snowflake sprite to Snowflake prefab for better visuals
• Check sorting layers if particles are hidden

SNOW SPAWNS IN WRONG PLACE:
--------------------------
• Snow spawns 1 unit above player by default
• Adjust spawn position in PlayerController.CastSnowSpell()
• Or adjust spawnHeight in SnowSpell prefab

=======================================================================
ADVANCED: MANUAL SETUP
=======================================================================

If you want to set up manually instead of using the tool:

1. Create the prefabs:
   • Create Snowflake GameObject with:
     - SpriteRenderer (white/cyan color)
     - CircleCollider2D (trigger, radius 0.15)
     - Rigidbody2D (kinematic)
     - Snowflake.cs component
   
   • Create FallingSnowSpell GameObject with:
     - SnowSpell.cs component
     - Assign Snowflake prefab to snowflakePrefab field

2. Save as prefabs in /Assets/Prefabs/

3. Assign to Player:
   • Select Player in scene
   • Find PlayerController component
   • Drag FallingSnowSpell to "Snow Spell Prefab" field
